## About the Project

This project is a front-end implementation of a company landing page for AITB Academy, a technology training and solutions provider. It was built from scratch using only HTML and CSS — no frameworks or libraries.

The page includes four sections, each targeting a specific set of front-end skills:

| Section | Skills Demonstrated |
|---|---|
| Navbar | Flexbox layout, sticky positioning, hover states |
| Hero | Background image, overlay, responsive typography |
| Our Solutions | CSS Grid, card components, responsive breakpoints |
| Testimonials | CSS Grid, card layout, multi-column responsive design |

---

## Built With

- **HTML5** — semantic structure and accessibility best practices
- **CSS3** — Flexbox, Grid, media queries, transitions, hover effects
- No JavaScript frameworks · No CSS libraries · No dependencies

---

## Features

- ✅ Fully responsive — adapts to desktop, tablet, and mobile
- ✅ Sticky navigation bar
- ✅ Hero section with dark overlay for text readability
- ✅ CSS Grid solutions section with hover card effects
- ✅ Testimonials section with responsive grid collapse
- ✅ Smooth button and card hover transitions
- ✅ Valid HTML5 with correct viewport meta tag

---

## Responsive Breakpoints

| Breakpoint | Layout |
|---|---|
| > 1024px | 3-column grid (Solutions & Testimonials) |
| 769px – 1024px | 2-column grid |
| ≤ 768px | Single column, stacked layout |

---

## What I Learned

- How to structure a multi-section web page with semantic HTML
- Using **CSS Flexbox** for navigation and inline layouts
- Using **CSS Grid** for complex multi-column content layouts
- Writing **media queries** to make layouts fully responsive
- Applying **CSS transitions** and hover effects for interactivity
- Layering background images with overlays using `::before` pseudo-elements
- Using `position: sticky` for persistent navigation
